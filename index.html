Installation Guide: iloader

iloader allows us to create a pairing file for programs like StikDebug to talk to your device remotely. This pairing file is device-specific and required for core functionality.

Warning Caution: If you update or reset your iDevice, your pairing file will expire and you will need to replace it. This also occurs at random times due to Apple's software architecture.

Prerequisites

macOS

  • Download and install iloader for Mac.

Windows

  • Download and install iTunes.
  • Download the iloader installer (exe or msi).
  • Run the installer.

Linux

  • Install usbmuxd (use your package manager).
  • Download iloader: Debian, Fedora, or AppImage.

Placing Your Pairing File

  • For best results, connect your iDevice to your computer via USB (wireless may work if the device appears).
  • Launch iloader and click Manage Pairing File.
  • To the right of "StikDebug" (and any other apps you wish), click Place.
  • Wait for the message: "Pairing file placed successfully!"
